Filter the data with the following options:

  • Field: user can select to filter the analysis by Topic, Sentiment, or Emotion.
  • Operator: currently, only the operator “IS” has been implemented, which indicates that both field on either side of the operator must be true to produce a result.
  • Value: user can select desired topics for analysis. Selecting the Domain will auto select all topics within the domain.

Click the "+" icon next to the filters to add another filtering row. Click on the trash can icon to delete the respective filter row.

Once filters have been selected, click "Apply" to update the visualizations on the selected tab. To clear all filters, click "Reset".
