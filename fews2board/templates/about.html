About

The periodic country analyses elaborated by FEWS NET have enabled the creation of a comprehensive historical database comprising consistent triannual reporting on over two dozen countries. The longitudinal breadth of this dataset has the potential to provide information and insights into long-term trends regarding shocks over time, coping capacity, and livelihoods.

Systematically assessing this documentation from a historical perspective not only provides a comprehensive overview of food insecurity in FEWS NET covered countries, but also provides an additional evidence-base to support analysis.

Machine-driven content analysis techniques stand as effective approaches to make sense of all the data available, as extracting knowledge from extensive text-based sources through conventional approaches is a challenge. In particular, text mining seeks to automate the retrieval of high-quality information from text by finding patterns and trends through machine learning, statistics and linguistics. The aim of this technique is to enable the analysis of large amounts of unstructured text.

Using publicly available reports from the FEWS NET website, this interactive dashboard presents results from an integrated machine learning model that systematically analyzes textual documents to derive insights.

The analytical model employs supervised and unsupervised text mining to selected FEWS NET documentation. A custom analytical framework was developed considering the dimensions of FEWS NET’s framework for food insecurity, from which analysis detects historical trends, associations, and thematic dynamics, as represented in the documents.

The main questions this dashboad intends to explore are:

  1. What historical trends (or anomalies) can be identified regarding the prevalence of various dimensions from the FEWS NET framework for food insecurity?
  2. How do trends compare by region and type of indicators?
  3. How are the dynamics, such as the interactions within and between the different dimensions, represented? How do these representations transform in time and space?

With data since 2000, the dashboard is integrated with FEWS NET's reporting system to update as new documentation is published.
